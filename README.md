# Project Title

PDF Reader

## Description

Program to read all the PDF's from an excel file, and gives a report on which of these PDF's are actually downloadable.

## Getting Started

### Dependencies

* requests
* XlsxWriter
* openpyxl
* pandas
* tqdm

### Installing

* Download either complete folder og docker image.

### Executing program

* git clone -project-
* cd -project-
* python src/pdf_reader.py


## Help

Remeber to close the Data/Downloaded_pdfs.xlsx after viewing if running again.
